# 自動安裝LibreNMS腳本

## Install_Ubuntu
系統資訊:  
OS version: Ubuntu 20.04 LTS  
PHP version: 7.4  
Web: Apache2  
Database: MariaDB  
Database account: librenms/password  
timezone: Asia/Taipei  

安裝步驟:  
```
git clone https://github.com/Paul1014/Automation-Kit
cd ./Automation_Kit/LibreNMS_install
sudo sh install_Ubuntu
```
